# 📊 MIS Reporting & Analytics

## 📌 Overview

This repository is a structured **MIS (Management Information System)** showcasing Excel-based reporting, dashboards, and data analysis across multiple business departments including HR, Finance, Sales, Customer CRM, Operations, and Advanced Excel analytics.

The goal of this portfolio is to demonstrate practical **business reporting, data analysis, and dashboarding skills** required for MIS Executive and Data Analyst roles.
